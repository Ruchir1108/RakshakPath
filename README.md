# 🚦 RakshakPath – AI-Powered Road Safety & Emergency Response Analytics

## 📌 Project Overview

RakshakPath is an AI-powered geospatial analytics platform developed to improve road safety and emergency response management. The system analyzes historical accident data from the Karnataka State Transport Department to identify accident-prone “Black Spots” using the DBSCAN clustering algorithm.

The platform provides an interactive dashboard for accident heatmap visualization, infrastructural risk analysis, emergency facility detection, and AI-based safe route prediction. It helps urban planners, emergency services, traffic police, and daily commuters make data-driven safety decisions.

---

## ✨ Features

* Accident hotspot detection using DBSCAN clustering
* Interactive geospatial heatmap visualization
* AI-based safe route prediction
* Route safety scoring system
* Nearby hospital and police station locator
* Emergency response monitoring dashboard
* Temporal filtering based on weather and time conditions
